Pixiv 下载器！

发表于 2020-05-16 分类于 website

之前练手 Vue 一直写一些奇怪的东西，往服务器上放也没有很大意义那种，于是就突然想搞一个有用点的网站（其实刚写完以后才知道有 pixez viewer，我死了，就权当是给 ios 用户提供的吧）

还是没用上 webpack，感觉这种小网站一般还是不 pack 写的顺一点，虽然这个逻辑已经比较复杂了。爬 pixiv 稍微有点麻烦，最开始莫名其妙 403，发现需要把 Header 改成特定的样子，非常隐蔽（

用法大概就是可以搜索、也可以直接输入 id 下载。点击标签会自动跳转，点击图片会弹出下载框，可能有一堆分p。之后可能还会加一些搜精选之类的功能，就看放假啦……

放个链接：http://hyz18.top/pixiv
