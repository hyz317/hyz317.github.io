离散平面图总结

发表于 2020-05-16 分类于 discrete

平面图

若一个图能保证没有两条线相交地画在平面上，称为平面图

平面图中每一个内部不含节点和边的区域均为域或面

两个域相邻 <=> 有公共边界

一条边是两个域的公共边界 <=> 这条边不是割边

平面图外围所有地方称为无限域，其他域为内部域

域的边界数可能为 1（自环）或 2（重边）

若连通支 m < 9 或 n < 5，一定是平面图

欧拉公式

平面联通图中域的数目 d = m - n + 2

平面非联同图中域的数目 d = m - n + k + 1

若没有割边，域的边界数对域求和为 2m

极大平面图

满足：

1、不存在割边

2、连通

3、所有域的边界数均为 3（3d = 2m）（常用证明）

可以与 度数之和 = 2m 连用

推论：

1、极大平面图的 m = 3n-6，d = 2n-4（欧拉公式 + 性质3）(若为简单平面图，则是充要条件)

2、简单平面图的 m <= 3n-6，d <= 2n-4（欧拉公式 + 性质3，割边不增加域）

3、简单平面图中含有度小于 6 的节点

非平面图

典型的两个非平面图：K5（记作K(1)）和K3,3（记作K(2)）

向这两种图中插入任意 度为1 / 度为2 的节点，仍为非平面图。

一个图是非平面图 <=> 这个图有K1或K2型子图

图的平面性检测

0、看到割点就将图分成两个图，看到割边就删掉

1、若有多个连通支，挨个检测

2、看到自环就删掉

3、看到度为2的节点就删掉，并补上一条边

4、看到重边就删掉

a. 若m<9或n<5，则G是可平面图

b. 若m>3n-6，则G是非平面图

c. 不满足a和b，需要进一步测试

对偶图

G 的对偶图记作 G*

画对偶图的过程：

1、每个域整一个节点，无限域一共算一个节点，自环和重边都算

2、如果两个域有一条公共边，这两个节点画一条边

3、如果一条边只是一个面的边界（即割边），绕这个面对应的节点整一个自环

注意！是针对于边的！每条边对应对偶图的一条边！

对于平面图，对偶图存在且唯一（充要）

对于平面连通图，对偶图的对偶图是原图

平面连通图的对偶图：n* = d，m* = m，d* = n

对偶图中某条初级回路 <=> 原图对应边的割集

证明题套路！

1、每个点的度大于6 -> 6n <= 2m

2、每个域的边界大于5 -> 5d <= 2m

3、欧拉公式

4、极大平面图 m = 3n-6，d = 2n-4

5、简单平面图 m <= 3n-6，d <= 2n-4

6、简单平面图无割边 -> 3d <= 2m

补充

简单图中至少有两个度相同的点
